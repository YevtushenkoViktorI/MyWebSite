# myPortfolioSite

## Start

```bash
npm install
npm run dev
```

## Font files

Put local fonts into `public/fonts`:
- `Sora-SemiBold.woff2`
- `Sora-Bold.woff2`
- `Manrope-Regular.woff2`
- `Manrope-SemiBold.woff2`

## Structure

- `src/styles/colorPalette.css` - one file with all colors
- `src/styles/localFonts.css` - local font setup
- `src/i18n/locales/` - separate localization files per language
